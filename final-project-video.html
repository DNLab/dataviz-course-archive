--- layout: default --- Assignment: Final Project Video

Please create a video presentation of your final project, after having completed the final finishing touches.

This should be no more than 8 minutes, and should capture the essence of the project.

As you've already written the script for this, the work consists of

  • Recording - Mainly screen capture, but a bit of your face would be nice for the opening and closing at least
  • Editing - Please do basic editing (if required), such as removing silences and false starts

This video, along with the project source code and deployed program, will be your main achievement from this course, which you can show off in your professional portfolio.

Please upload the video to YouTube and submit the link to the video as the submission for this assignment.

Please also link to the video from your project README (optional, but highly recommended).

Please also put a link from the video to the project.

 

  • For making videos, I'd recommend the following software:
    • Screen capture software such as OBS (if you're on Mac, QuickTime works well for this).
      • (optional) This has an option for a webcam face overlay in the corner.
    • (Optional) Video editing software such as KDEnlive
      • for editing out silences and false starts.
    • If you find another tool that works for you, please share it with the class (there's so much out there).
  • Please include your bl.ocks.org profile page as the first scene of the video.
    • Please add an image to your GitHub profile, it makes your bl.ocks.org profile look more professional.
