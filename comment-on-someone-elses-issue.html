Assignment: Comment on Someone Else's Issue

Everyone's project tasks will be flowing into Slack as GitHub issues links.

Please open someone else's list of issues and look through it, then make a comment on one of the issues.

The nature of the comment could be something like:

  • I'm facing a similar issue <paste link>, I wonder how you will solve this.
  • I've seen this solved in a bl.ock actually, it's here <paste link>
  • This is such a cool idea!
  • This issues seems a bit to large, perhaps better to split it into multiple issues.
  • This issue doesn't have enough of a description to really know when it's done.

Please submit the link to the issue in Canvas as the submission, and also please post the issue link to Slack, including any comments that make it unique or interesting.
